# Журнал расходов стройки

Простое локальное веб-приложение для учёта расходов на строительство дома.
Работает прямо в браузере: один HTML-файл, никаких серверов и установок.
Все данные хранятся в `localStorage` браузера, для надёжности предусмотрена выгрузка/восстановление JSON-резервной копии.

## Возможности

- Журнал расходов с категориями и подкатегориями (например, «Фундамент / Материалы»).
- Договоры по виду работ: сумма, срок оплаты, примечание.
- Автоматический расчёт по каждой категории:
  - оплачено по договору и остаток,
  - доп. расходы вне договора (позиции с подкатегорией),
  - **доля категории в общей сумме стройки** (отображается перед названием).
- Итог `Итого стройка` — сумма всех расходов.
- Группировка таблицы расходов **по категориям** с подитогами по каждой категории и общим итогом.
- Редактирование суммы, категории, подкатегории и комментария у любой записи.
- Резервные копии: выгрузка JSON и восстановление из файла.

## Запуск

Никаких зависимостей. Открыть в браузере файл:

```
Расходы стройка.html
```

Рекомендуется Chrome / Edge / Firefox актуальных версий.

## Хранение данных

- Записи расходов: `localStorage` → ключ `build_costs`.
- Договоры по категориям: `localStorage` → ключ `build_contracts`.
- Регулярно скачивайте резервную копию кнопкой «Скачать резервную копию» — это единственный способ переноса данных между устройствами/браузерами.
