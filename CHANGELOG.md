# История изменений

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
проект придерживается [Семантического версионирования](https://semver.org/lang/ru/).

## [1.1.0] — 2026-05-11

### Добавлено
- Файлы полноценного репозитория: `README.md`, `LICENSE` (MIT), `CHANGELOG.md`, `.gitignore`.

### Изменено
- Заголовок «Всего потрачено» переименован в **«Итого стройка»**.
- Доля категории в общей сумме теперь отображается **перед** названием
  категории, например `(42.6%) Участок: 1 200 000,00 ₽`.
- Таблица расходов внизу страницы сгруппирована **по категориям** вместо
  плоского списка по датам. По каждой категории выводится подитог,
  а в самом низу — общий итог стройки.

### Исправлено
- По категориям с договором (например, «Фундамент») доля в общей сумме
  не считалась — теперь рассчитывается корректно с учётом всех позиций
  (включая позиции с подкатегорией, такие как «Материалы для фундамента»
  и «подача бетона. насос»).
- Под категорией с договором отдельно показываются доп. расходы вне
  договора (если они есть).

## [1.0.0] — 2026-05-04

### Добавлено
- Базовая версия журнала расходов стройки в одном HTML-файле.
- Учёт расходов по категориям и подкатегориям.
- Договоры по видам работ с расчётом остатка и срока.
- Локальное хранение в `localStorage`.
- Выгрузка и восстановление резервной копии в формате JSON.
